# Getting Support

Synapse is a young open-source project. Here's how to get help.

## 🐛 Bugs

Open an issue: https://github.com/arajgor1/synapse/issues/new/choose

Please include:
- Synapse version (`python -c "import synapse; print(synapse.__version__)"`)
- Which framework adapter (if any) you're using
- A minimal reproducible example
- The full error / traceback

## 💡 Feature requests

Same issue tracker, with the `enhancement` label:
https://github.com/arajgor1/synapse/issues/new/choose

## ❓ Questions

Three options, in order of fastest-response:

1. **GitHub Discussions** — for design questions, "how do I…" questions,
   and architecture discussions: https://github.com/arajgor1/synapse/discussions
2. **Email** — direct line to the maintainer:
   aadityarajgor27@gmail.com
3. **Discord** — community chat (coming soon — join the waiting list via
   the repo header)
