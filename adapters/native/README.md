# adapters/native

Native framework adapters — the production adapters live at
[`sdk-python/synapse/frameworks/`](../../sdk-python/synapse/frameworks/).

Currently shipped (v0.2.8):

- AutoGen (Microsoft)
- CrewAI
- LangGraph (LangChain)
- smolagents (HuggingFace)
- Agno
- LlamaIndex
- Pydantic AI
- OpenAI Agents SDK
- Google ADK
- Hermes (Synapse-native)
- OpenClaw (TypeScript, in `sdk-typescript/`)

For the current roadmap and upcoming adapters, see
[`docs/roadmap/README.md`](../../docs/roadmap/README.md).
