# Quickstart — audit a trace

```bash
pip install synapse-protocol-py
synapse audit ./traces.json
```

Synapse auto-detects the format. Supported: OpenInference / OTel, LangSmith, AWS Bedrock, GCP Vertex Agent Builder, Azure AI Agent Service, generic JSONL.

For per-segment guides see [solo devs](../for-solo-devs.md), [small teams](../for-small-teams.md), or [enterprises](../for-enterprises.md).
